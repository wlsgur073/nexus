## 변경 사항

### @nexus/ui (packages/ui)
- shadcn/ui 컴포넌트 9개 추가: Tabs, Select, Textarea, Dialog, Alert, Checkbox, Table, Label, Skeleton
- base-nova 스타일 준수 (asChild 미사용, render prop 패턴)

### @nexus/codex-models (solutions/codex/models)
- 19개 엔티티 TypeScript 타입 정의 (standard, governance, validation, common-code, system, db-connection)
- 17개 enum 타입 (StandardStatus, RequestStatus, DraftStatus, UserRole, TargetType 등)
- 14개 API 모듈 (Mock 데이터 반환): auth, dashboard, explorer, standard-words/domains/terms, requests, approvals, inline-governance, drafts, comments, ai

### @nexus/codex-shared (solutions/codex/shared)
- 상수 4개: routes(사이드바 경로), menu(역할별 접근 매트릭스), status(3종 라벨+색상), query-keys(TanStack Query)
- 유틸 4개: physical-name(물리명 생성), status-color(Badge 색상), role-check(권한 체크), request-no(신청번호 파서)

### @nexus/codex-web (solutions/codex/web)
- 기반 레이어: QueryProvider, AuthProvider, middleware(인증 가드), hooks(useAuth, useDebounce, useDraftAutosave), lib(api, validators)
- 공통 UI 컴포넌트 7개: StatusBadge, TargetTypeBadge, DataTable, Pagination, EmptyState, Stepper, SimilarityBar
- 대시보드 컴포넌트 5개: StatCard, ActivityTimeline, ApprovalKpiPanel, RequesterDashboard, ApproverDashboard
- 표준 관리 컴포넌트 8개: ExplorerFilters, ExplorerTable, StandardDetailSheet, InlineEditForm, ChangeDiffTable, DeleteImpactStepper, NewStandardForm, AiSuggestionPanel
- 승인 컴포넌트 4개: ApprovalList, ApprovalDetailPanel, ApprovalActionForm, BatchActionDialog
- 레이아웃: CodexSidebar(역할 기반 메뉴 필터링)
- 라우트 7개: login, dashboard(역할분기), standards(탐색기+Sheet), standards/new, approvals, error, not-found
- 신규 의존성: @tanstack/react-query, react-hook-form, zod, @hookform/resolvers, sonner, nuqs
